Short definition about html css and java script

.HTML(Hyper Text Markup Language):

- Html stands for Hyper text markup language.
- Html is the standard language for web pages.
-With html you can create your own website.
-HTML elements are represented by tags.
-Html consist of a series of elements.
-Browser do not display the html tags but use them to render the content of the page.

EXAMPLE:

< html >
< head >
< title > This is my page< /title >
< /head >
< body >
< b > AShish kc< /b >
< p > Welcome < /p>
< /body >
< /html >

CSS(Cascading Stylesheet):

- CSS stands for Cascading Style Sheet.
- CSS describes how html elements should be displayed.
- CSS is a language that describes the style of an html document.
- CSS saves a lot of work. It can control the layout of multiple web pages all at once.
- External stylesheets stored in CSS files.
EXAMPLE:

body{
margin:0px;
padding:3px;
background-color:red;
}
.box{
font-family:cursive;
float:right;
color:#8888;
}



JAVA SCRIPT:
- Java script is the programming language of HTML and the web.
- Java script is easy to learn.
- JAva script can change HTML content.
- Java script can change HTML styles.
- Java script can change HTML attributes.
EXAMPLE:

< html >
< head >
< title > This is my page< /title >
< /head >
< body >
< inputonclick="call_fun();" value="click here" input type="button">
< script >
function call_fun(){
document.write("Hello world");
}
< /script >
< /body >
< /html >
